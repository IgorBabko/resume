Кратко о себе

Я родился в городе Конотоп Сумской области в 1994 году. Со времен младшей школы у меня было большое желание связать мою будущую деятельность с компьютерными технологиями.

Программированием начал заниматься с первого курса университета (2012 год). Первые два курса изучал различные языки (С/С++, Java, C#, JavaScript). На третьем курсе уже точно определился со своим будущем направлении в программировании - web development.

С тех пор я освоил следующие технологии: HTML (Jade), CSS (Sass, Stylus, PostCSS), JavaScript (ES6, CoffeeScript, jQuery, VueJS, NodeJS + Express, Angular 2 + TypeScript, MeteorJS), Twitter Bootstrap, Gulp, PHP + Laravel. На сегодняшний день упорно продолжая заниматься разработкой приложений для web.
